Quick Start

  1. Open Home and choose the correct DWO form.
  2. Fill patient details, diagnosis, physician details, and ordered items.
  3. Use the built-in search or saved-record tools if the patient or doctor already exists in your account.
  4. Save the record to keep it searchable for future forms.
  5. Print or Save PDF once the checklist is complete.

Best Daily Workflow

  1. Check Saved Records first if the patient was handled before.
  2. Use New From This when you want the same patient and doctor on a different DWO type.
  3. Use recent or pinned patients/doctors for one-click reuse when the same names come up often.
  4. Save the new DWO immediately after reviewing it so it stays in your personal library.

What Saves Time Most

  • Inline patient and doctor suggestions while typing names.
  • Quick Fill buttons inside forms for patient, doctor, or full saved-record import.
  • Recent and pinned shortcuts on the Saved Records page.
  • Automatic quantity defaults for supported form-specific code rules.

Your profile controls the identity details that appear on fax cover output. Set that once in Profile and keep it updated.

Working Inside a Form

  • Patient and doctor smart suggestions: start typing a patient name or physician name and the form will search your saved DWOs for matches.
  • Quick Fill section: use Find Patient, Find Doctor, or Use Saved Record when you want to pull information from an existing saved DWO.
  • Carry-forward behavior: the app remembers useful form-level preferences like the fax cover subject and usual physician details by form type, plus patient-linked insurance details when name and DOB match.
  • Checklist protection: the form highlights missing key fields before saving or printing so incomplete DWOs are easier to catch.
  • Autosave drafts: while you type, the current form autosaves locally so accidental refreshes are less painful.
Tip: when a patient already exists in your account, it is usually faster to reuse their details from a saved DWO than to type them again manually.

Saved Records Page

  • Search deeply: search can find records by patient, doctor, insurance, HCPCS, diagnosis, phone, DOB, and form type.
  • Open Record: use this when you want to continue editing the same saved DWO.
  • New From This: use this when you want a fresh DWO that starts with the same patient and doctor information from a saved record.
  • Recent Patients / Recent Doctors: these let you start a new DWO using commonly reused people from your own account history.
  • Pinned Patients / Pinned Doctors: pin frequent names so they stay easy to reach without searching every time.

Keyboard Shortcuts

Ctrl/Cmd + S

Save the current DWO to cloud records.

Ctrl/Cmd + P

Print or save the current DWO as PDF using the form's print output.

/

Open patient lookup quickly when you are not typing inside a field.

Alt + Shift + P

Open the patient quick-fill search directly.

Alt + Shift + D

Open the doctor quick-fill search directly.

Enter

Select the first autocomplete suggestion or move to the next visible field while filling out the form.

Printing and PDFs

  • The app uses the patient name, form name, and current date to create cleaner PDF/browser titles.
  • Use Print/Save PDF only after the checklist is clear so the final output is ready to send.
  • If you need a fax cover, make sure your profile is correct and physician fax details are filled in.

Profile Page

  • Set your display label, first name, last name, and extension.
  • These details support your fax-cover identity and staff-specific workflow.
  • Each staff account works with its own saved records, shortcuts, and tracking.

Best Practices to Get the Most Out of the App

  • Save every completed DWO, even if it seems one-time. Future reuse is where the app becomes much faster.
  • Pin high-volume doctors and common repeat patients so they stay one click away.
  • Use New From This when the patient and doctor stay the same but the form type changes.
  • Keep physician fax numbers and profile details accurate so print and fax packages come out clean.
  • Let the form defaults help you, but always review editable fields before finalizing.
Simple rule: if the information already exists in one saved DWO under your account, try to reuse it instead of typing it again.
